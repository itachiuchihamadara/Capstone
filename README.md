# eComBot Final

End-to-End eComBot System — Ship a Production-Ready Multi-Agent Platform

This is the final completed capstone project for the Google ADK.

## Setup

1. Copy `.env.example` to `.env` and fill in your keys.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python src/ui/app.py
   ```

## Architecture
- Orchestrator Agent
- Support Agent
- Sales Agent

## Features
- RAG using ChromaDB
- FastMCP External Integrations
- ReAct Reasoning Loop
- Voice Interface
- LangSmith Tracing
- Guardrails for Security
